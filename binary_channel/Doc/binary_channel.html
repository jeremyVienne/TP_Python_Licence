Binary channel¶

Implementing a binary symmetric memoryless channel (CBSSM)

author:FIL - IEEA - Univ. Lille 1 (nov 2010, août 215)
binary_channel.noise(p)¶

Return a 1-bit with probability p

Parameters:p (float) – Probability of having a 1
Returns:A bit
Return type:int
UC:p >= 0 and p <= 1
Examples: 
>>> noise(0)
0
>>> noise(1)
1
binary_channel.cbssm(p, byte)¶

Transmit a byte through a CBSSM and return the received byte.

Parameters:
  • p (float) – Probability of having an error in the CBSSM
  • byte (int) – The byte to be transmitted
Returns:

byte where each bit was modified with a probability p
Return type:

int
UC:

p >= 0 and p <= 1 and byte >= 0 and byte < 256
